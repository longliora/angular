### cài node js và angular cho linux
1. Cài NVM
```curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash```

    Reload shell:
    ```source ~/.bashrc```
2. Cài Node LTS
   
    ```nvm install --lts```
    
    ```nvm use --lts```

# Cài Angular CLI

     ```npm install -g @angular/cli```
     
  kiểm tra: 
    
    ```ng version```
